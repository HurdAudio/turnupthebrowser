How to Web Audio API:

const audioContext = new (window.AudioContext || window.webkitAudioContext)();

The Web Audio API is constructed as an audio routing graph:

This allows for arbitrary routing between AudioNodes with inputs and/or outputs.

A Source Node has no input and a single output.

A Destination Node has a single input and no output.

Other AudioNode types may pass through a signal, altering it in a defined manner. These have both inputs and outputs.
