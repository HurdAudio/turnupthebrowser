Web Audio API - The Synthesizer Embedded in Your Browser

  • High-level API for synthesizing and processing audio in web applications

  • A modular approach to generating or handling audio output

  • API currently has a feature set that enables a wide range of applications:

    • Emulating synthesizer architecture

    • Mixing multiple streams of audio

    • Dynamic spatialization of audio signals

  • All of this functionality is literally just sitting under the hood - no external resources required (give or take a set of speakers or headphones)
